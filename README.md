# eZ Mailjet Api V3
eZ Mailjet API V3 is an eZ Publish extension for generic user registration.
The current version of eZ Mailjet API V3 is 1.0.
More to come soon.

## Requirements
- Mailjet API Key, Secret Key and List ID
- PHP (v. >= 5.4)
- eZ Publish 4.x
- jQuery 1.11.x
